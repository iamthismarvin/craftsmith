## Features

```
- Simple RPG mechanics.
- Fight in dungeons and gain loot!
- Upgrade your weapons through enchantment!
- Mobile Friendly.
- Install on your device as a Progressive Web App and play offline!
```

## Stack

Vue.js, TypeScript, Tailwind CSS, and IndexedDB.
